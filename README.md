# my_first_repo_in_mai
Демонстрация работы в гите ЯЛ 2025
## Основные функциональные требования
1) Пользователь сможет создавать новые записи о своём настроении каждый день и просматривать свои старые записи

2) Оценка настроения и состояния
Программа будет последовательно собирать информацию о настроении пользователя через несколько простых окон:

Окно выбора настроения: 5 смайликов (радостный, злой, нейтральный, грустный, уставший)

Окно причины настроения(пользователь вводит в текстовое поле свою причину выбранного настроения или "кратко о сегодняшнем дне"

Окно комментария(текстовое поле для подробного описания чувств)

Окно оценки дня: от 1 до 10

3) Статистика и анализ за неделю(В конце недели программа проанализирует все записи и даст полезные выводы)

Показывает самое частое настроение за неделю

Выдает советы на основе преобладающего настроения

Рассчитывает среднюю оценку недели

4) Система советов (Программа будет давать персонализированные советы для улучшения настроения)

Если больше радостных дней, то (например)"Продолжайте в том же духе! Вы большой молодец"

Если больше грустных дней, (например) советы: "Погуляйте на свежем воздухе", "Пообщайтесь с друзьями", "Побалуйте себя чем-то вкусным"

Если больше уставших дней,  советы:  "Выспитесь", "Отвлекитесь от дел на час"

( Все данные будут сохраняться и их можно будет экспортировать)

Автоматическое сохранение в базу данных

Экспорт недельной статистики в TXT файл

Резервное копирование данных(?)Архитектура и дизайн
Основные модули
Главное окно (MainWindow): Меню и навигация

Последовательных окон (WizardManager): Управляет пошаговым процессом ввода данных (настроение,  причина, комментарий, оценка)

Базы данных (DatabaseManager): Отвечает за сохранение и загрузку всех записей

Статистики (StatsCalculator): Анализирует данные и выдает статистику за неделю

Советы (AdviceGenerator): Выдает персонализированные советы на основе данных

(ExportManager): Сохраняет данные в TXT файлы

Структура базы данных
Количество таблиц: 2 таблицы(Запись настроения( дата записи, выбранное настроение (смайлик), причина настроения,  комментарий пользователя, оценка дня от 1 до 10) и Советы)

Структура меню и панели инструментов
Новая запись

История записей

Статистика за неделю
